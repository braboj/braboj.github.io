# 👋 About

I'm Branimir Georgiev, a Senior Python Developer and former Industrial 
Automation Engineer with over a decade of experience in software engineering, 
automation systems, and advanced control strategies.

My journey began in automation, working on large-scale industrial systems 
with Siemens and Honeywell technologies. Over time, I transitioned into 
software development — where I now specialize in Python, data analysis, 
network protocol testing, and agile software delivery.

- **Software Engineering** – Python, C, Java, Dart, C#, Visual Basic
- **Industrial Protocols** – MQTT, Modbus, DeviceNet, Profibus, Profinet
- **Automation & Control** – Advanced PID tuning, Fuzzy Logic, HMI design
- **Testing & QA** – Pytest, Scapy, Jenkins, GitHub CI, Docker
- **Methodologies** – Agile, SAFe 6 Practitioner, mentoring & documentation
